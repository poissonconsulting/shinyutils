# Navbar css

Navbar css

## Usage

``` r
css_navbar(
  text_color = "#E0E0E0",
  text_selected_color = "#37CBA2",
  text_size = "15px",
  background_color = "#010101",
  float = "right"
)
```

## Arguments

- text_color:

  A character string of navbar text color.

- text_selected_color:

  A character string of selected navbar text color.

- text_size:

  A character string of navbar text size.

- background_color:

  A character string of navbar background color.

- float:

  A character string indicating position of tabs in navbar ('left',
  'right' or 'center')

## Value

a shiny tag.
